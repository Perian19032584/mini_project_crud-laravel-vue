# Установка
                     
#### 1. Скачать файл
##### 2. Ввести команду: `composer install`
##### 3. Ввести команду: `npm install`
##### 4. Ввести команду: `php artisan laravel-vue:install`
##### 5. Ввести команду: `php artisan key:generate`
##### 5. Ввести команду: `php artisan migrate`
